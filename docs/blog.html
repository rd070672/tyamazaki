渡米研究での経験と学び

2023-08-21
2020年3月から1年間、私は米国東海岸のメリーランド大学で在米研究に取り組みました。この期間、新型コロナウイルスが拡大し、その影響で起こったさまざまな社会的変化をアメリカで直接経験しました。異文化に触れる日々は新しい視点や挑戦をもたらし、日本とアメリカの研究文化の違いを深く理解することができました。

大学での仕事内容

2023-08-21
大学での仕事内容についてまとめたいと思う。嘱託助教（実態はプロジェクト研究員）として、どんな仕事内容に取り組んでいるか、どんな成果が求められているのか。大学雇用の助教でないため、大学や授業関係の仕事がほとんどないため、研究活動に全振りできる最高の期間である。

5分で作れるホームページ

2023-08-21
GitHub pagesを利用して、サーバー費用無料でホームページがあっという間に作成できる。ドメイン名さえ、年3000円で取ってしまえば、完全オリジナルなウェブサイトが作成できる。しかも、GitHub pagesは、GitHubにアップロードしたファイルをそのままホームページとして公開できるので、とても便利である。

ChatGPT自動解説botで論文を毎日読もう

2023-08-21
Slackのチャンネルを作成し、そこにChatGPTで翻訳・要約したARXIV論文を自動で送り続けることで、研究者の情報収集を効率化することができる。論文要約だけでなく、キーワード解説、この論文に記載のない発展課題まで説明してくれるプロンプトです。ArxivからでもNature論文からでもAPIで引っ張ってこればslackで解説させられます。
